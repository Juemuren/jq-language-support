## Install

### From extension marketplace

Not yet

### From source code

```sh
# Clone the repository
git clone https://github.com/juemuren/jq-language-support.git
cd jq-language-support
# Package
npx @vscode/vsce package
# Install
code --install-extension jq-language-support-*.vsix
```
